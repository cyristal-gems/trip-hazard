# Trip Hazard

**Trip Hazard** is a desktop arcade runner where a pixel duck dashes through a bright marsh scene, hopping over bushes and ducking under swooping birds. Inspired by the Google Chrome dinosaur runner, the game uses simple controls, steady motion, and increasingly tricky obstacles.

## 🎮 How to Play
- Press **Space** or **↑** to hop over bushes.
- Press **↓** to duck under birds, or hold it while airborne to drop faster.
- Press **P** or click the pause button to pause the game.
- After a game over, click the centered **↻** arrow to restart.

---

## 🚀 Features
- Bright marsh scene with a blue sky, soft clouds, reeds, rolling greenery, and a grassy path  
- Animated yellow pixel duck with a playful hop, orange bill, flapping wing, and small landing puffs  
- Leafy green bush obstacles and light blue bird sprites that match the marsh setting  
- Compact score and pause HUD, plus a centered circular restart arrow after game over  

---

## 🧰 Tech Stack
- **HTML** — Structures the single-page game, including the header, canvas stage, scoreboard, pause button, start button, and control notes. 
- **CSS** — Handles the responsive layout, marsh-inspired color palette, HUD styling, buttons, and mobile-friendly stage sizing. 
- **JavaScript** — Runs the game loop, keyboard controls, jumping, ducking, fast-drop behavior, scoring, speed ramp, obstacle spawning, collision detection, and centered restart state. 
- **Canvas API** — Draws the animated duck, moving marsh background, bushes, birds, dust puffs, ground, and path frame by frame.
